Resumo

Básicamente o que foi feito nesse site, foi apenas para demonstrar como é o método Inline. Em resumo o método Inline são todas as CSS aplicadas em cada tag específica. Um exemplo na tag "h1"

        
            <h1 style="background-color: rgb(98, 166, 243)>

O que basicamente foi feito nessa demonstração foi a aplicação das CSS na tag "h1", mas também podemos usar em outras diversas tags como a "body", "p" entre outras.

Em resumo o método Inline é a prática de aplicar as CSS em cada tag precisamente.(Configurações pontuais)

É indicado a utilização desse método apenas quando houver necessidade de aplicar em tags específicas, não sendo necessário fazer isso com várias tags.
