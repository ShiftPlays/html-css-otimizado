Resumo

Basicamente os Estilos Locais de CSS é uma outra metodologia da forma como usamos as CSS. Nesse caso as usaremos na tag "head"(<head></head>). Um exemplo: 

        
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport"> 
    <content="width=device-width,> 
    <initial-scale=1.0">
    <title>Estilos Locais</title>
    <style>
        body {
            background-color:blanchedalmond;
            font-family: Arial, Helvetica, sans-serif;
            font-size: 20px
        }
        h1 {
            color:darkblue;
            background-color:blueviolet;
        }
        h2 {
            color: darkred;
        }
        p {
            text-align: justify;
        }
    </style>
</head>

Cada tag descrita dentro da tag "style"(com base nos parâmetros de modificação.), será a forma como será representada no site.
