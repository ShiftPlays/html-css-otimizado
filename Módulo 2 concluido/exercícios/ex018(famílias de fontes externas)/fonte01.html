Capítulo 14 Aula 06 - Usando Google Fonts

Para aplicarmos as fontes com base no Google Fonts precisaremos primeiro acessar o site e escolher uma fonte. Tendo escolhido clique em "get font" e depois vá na sacolinha onde estará as fontes, nela está escrito ao passar o mouse "view select families" > "< > Get embed code" > "selecione o @import" > Copie as delarações da regra "@import" e coloque nas primeiras linhas dentro da tag "<style></style>" em "head"(a tag <head></head>)

Exite uma regra muito importante no CSS que é, tudo que começa com "@" tem que ser colocado debaixo da tag "style" - (<style></style>) nas primeiras linhas antes dos seletores e declarações.

Vou copiar um "@import" do google fonts para demonstrar de uma forma mais eficiente:

            
    <style>
    @import url('https://fonts.googleapis.com/css2?family=Roboto+Mono:ital,wght@0,100..700;1,100..700&display=swap');
    
        body{
            font-family: Roboto Mono, serif;
        }
        
        h1{
            
                font-family: Arial, sans-serif;
                color: blue;
            
        }
        
        p {
            
                font-family: Arial, sans-serif;
                color: blueviolet;
            
        }
    </style>

Nesse exemplo vemos como é simples modificar a fonte no CSS sem sequer ter a fonte no nosso computador

Tudo que começa com "@" no CSS seja qual for o estilo, é chamado de "Regra"
(Aqui é apenas um detalhe que temos que fixar na nossa cabeça e vou sempre demostrar a partir desse site aqui por que não nos atentamos por mais que na primeira apostila diga esse detalhe lá no módulo 1)

O nome que damos as tags que nos referimos dentro tag "style" são chamadas de seletor.

Todas as escrituras dentro dos colchetes "{}" provenientes dos seletores são chamados de declarações

Uma observação a se fazer é que o google fonts está em constante mudança, pode ser que já esteja completamente diferente quando você, vulgo eu do futuro, estiver lendo isso de novo, mas fica a observação aí a gente é foda e você sabe disso gostosão.
Agora uma outra observação esse mesmo exemplo que demosntrei está carregando a forma como eu modifiquei a fonte desse site, usando apenas google fonts.

Todas as fontes do google fonts são livres para uso.

Três a quatro estilos de fontes para nosso site é mais que o necessário, pois podemos dividir as fontes para os títulos, porém se passar disso pode fazer com que o site vire uma bagunça visual e não queremos isso.

Quando formos lançar nosso site, é bom que a gente use desse artifício para nosso site sair mais leve, e consequentemente melhor para as pesquisas assim como estudamos no primeiro capítulo.
