Conceito da tag "header"
(Copiado da internet apenas para estudar)

Conceito I

Em HTML, o elemento <header> é usado para definir o cabeçalho de uma página ou de um elemento dentro dela. Serve para apresentar informações introductorias ou navegacionais, como logotipos, títulos e navegação principal. O <header> pode conter vários elementos, como <h1> (o título principal da página), <nav> (para navegação), e outros elementos de estrutura.

Conceito II

  • O elemento <header> representa um cabeçalho introdutorio ou navegacional.
  • Pode conter informações introdutórias sobre o conteúdo, como o título principal (H1), subtítulos, uma lista de navegação ou um logotipo.
  • Cabeçalho principal da página: É comum colocar o <header> na parte superior da página, contendo o título principal do site e a navegação principal.
  • Cabeçalho de um artigo ou seção: Pode ser usado para definir um cabeçalho para um artigo ou uma seção específica dentro de uma página.
  • Cabeçalho de uma seção dentro de um artigo: É possível ter múltiplos <header> dentro de um mesmo artigo, cada um para uma seção específica.

Elementos a serem usados dentro de "header"

Elementos dentro de um <header>:

O <header> pode conter elementos como:

  • <h1> ou <h2>: Para títulos e subtítulos.
  • <nav>: Para a navegação principal da página.
  • <img>: Para inserir um logotipo.
  • <address>: Para informações de contato.
