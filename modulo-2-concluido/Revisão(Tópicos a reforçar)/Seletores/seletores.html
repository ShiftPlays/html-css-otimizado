id

No CSS, o "id" é usado para identificar um elemento HTML específico e aplicar estilos únicos a ele.

Class

No CSS, a classe (class) é um seletor que permite aplicar estilos a múltiplos elementos HTML que compartilham características em comum.

Pseudo-classes

: = pseudo-class

As pseudo-classes em CSS são seletores que permitem aplicar estilos a um elemento com base no seu estado ou posição em um elemento pai.

Existem várias pseudo-classes para estilos, podemos citar
:hover, :visited, :active, :checked, :empty e :focus.

  • :hover
    Em CSS, :hover é uma pseudo-classe que permite aplicar estilos a um elemento quando o cursor do mouse passa sobre ele.

  • :visited
    Em CSS, :visited é uma pseudoclasse que seleciona links que o usuário já visitou em uma página web.

  • :active
    Em CSS, :active é uma pseudo-classe que aplica estilos a um elemento quando o usuário está ativamente interagindo com ele, frequentemente usado para dar aos usuários feedback visual sobre a interação, indicando que eles estão clicando em um elemento.

  • :checked
    Em CSS, :checked é uma pseudo-classe que seleciona elementos <input> (caixas de seleção e botões de rádio) e <option> que estão selecionados ou "marcados". Permite aplicar estilos específicos a esses elementos quando estão no estado selecionado, sem a necessidade de usar JavaScript.
    A pseudo-classe checked é usada para estilizar elementos <input type="checkbox">(), <input type="radio">() e <option> quando estão selecionados. fonte

  • :empty
    Em CSS é usada para selecionar elementos que não possuem nenhum conteúdo, elementos que estão vazios (filhos/children, incluindo texto ou espaço em branco). É útil para estilizar(dar estilo) ou manipular elementos vazios de forma condicional.

  • :focus
    Em CSS é usada para estilizar(dar estilo) um elemento quando ele recebe o foco, geralmente ao ser clicado ou selecionado com a tecla Tab.

Pseudo-elementos

:: = pseudo-element

Em CSS, pseudo-elementos são palavras-chave que permitem estilizar partes específicas de um elemento HTML, que não são elementos de forma explícita no código HTML, como a primeira letra, a primeira linha ou conteúdo adicionado antes ou depois de um elemento.

Os principais pseudo-elementos usados nas CSS são
::before, ::after, ::first-letter, ::first-line.

  • ::before Insere conteúdo antes do conteúdo de um elemento, sem a necessidade de criar um novo elemento no HTML.
    Exemplo: ssss

  • ::after
    insere conteúdo após o conteúdo de um elemento específico, utilizando a propriedade content. Este conteúdo inserido é tratado como um filho do elemento, e não como um irmão do seu conteúdo.

  • ::first-letter
    É um pseudoelemento que seleciona a primeira letra do primeiro elemento de um texto de nível de bloco (como um parágrafo <p>) e permite aplicar estilos apenas a essa letra.   • ::first-line
    É um pseudoelemento que permite estilizar a primeira linha de texto dentro de um elemento de nível de bloco (como um parágrafo <p>). Com este pseudoelemento, é possível aplicar propriedades de fonte, cor, e fundo, entre outras, especificamente à primeira linha de texto dentro de um elemento.
