Lista de Definições

HTML
Linguagem de marcação para criação do conteúdo de um site.
CSS
Linguagem de marcação para criação de design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site

Todas as listas de definições estão dentro da tag "<dl> </dl>"(Definition List) Cada termo é um "<dt>"(definition term - fechamento opcional) e cada descrição é um "<dd>"(definition descripion - fechamento opcional)

Exemplo: 

        
        <dl>
            <dt>HTML</dt>
            <dd>Linguagem de marcação para criação do conteúdo de um site</dd>
            <dt>CSS</dt>
            <dd>Linguagem de marcação para criação de design de um site.</dd>
            <dt>JavaScript</dt>
            <dd>Linguagem de programação para a criação de interatividade de um site</dd>
        </dl>
