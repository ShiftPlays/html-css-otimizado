Na mesma linha de tag <a href="" > que aprendemos mais cedo vulgo as âncoras, no HTML5, essas mesma tag recebeu aguns atributos especiais para fazer download.

Atributos com o nome "download"(<a href="" download="" type="">) e "type"(<a href="" downlaod="" type="" >) foram adicionados para efetuar download com uma lista de media types para auxiliar nas modalidades de arquivos como .rar, .zip entre outros que se encontra facilmente no site da IANA.org

Esse site vai ajudar muito na necessidade de consultar as informações certas para o atributo "type" em caso de arquivos desconhecidos(exemplo: .rar)
