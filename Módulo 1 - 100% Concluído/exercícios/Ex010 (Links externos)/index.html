O mesmo exemplo de um jeito simplificado:

        
<p>Você pode acessar meu<a href="O link do canal no Youtube">Canal no Youtube</p>
<p>Você pode baixar<a href="Link do site para download" target="_blank" rel="external">The Sims 2</a></p>

Percebe-se que ao clicar no link escrito "canal no Youtube" o site é redirecionado para o Youtube sem deixar pistas do nosso site, isso acontece porque não acrescentamos na nossa tag "<a href="">" o complemento "<a href="O Link" target="_blank" rel="external">" que possíbilita qua se abra apenas uma outra guia mantendo nosso site aberto para uso.

O nome desses links que disponibilizamos no nosso site se chama "hyperlinks" e para criar "hyperlinks" devemos criar âncoras através da tag <a>("Anchor" - em português "âncora"). O atributo dessa tag é o "href"(<a href=""> - "hypertext reference" em português "referência hipertexto")

Target

Para poder controlar onde o site de destino vai abrir, podemos usar o atributo target pertencente a tag "a"(<a href="" target="">) que suporta os seguintes valores

  1. _blank:Vai abrir o link em uma nova janela em branco
  2. _self:vai abrir o link na janela ou frame atual(padrão)
  3. _top: vai desfazer todos os frames e abrir o destino no navegador completo(Cancela todos os frames e abre a nova página no mesmo navegador framename: Abre a página em um frame nomeado e especificado)
  4. _parent:abre a página destino na página anterior do histórico de navegação(Abre a página na mesma janela do link )

rel

O atributo "rel" pertencente a tag "a"(<a href="" target="" rel="">)indica qual é a natureza do destino, esse atributo aceita vários valores, entre eles:

  1. next:indica que o link é para a próxima parte do documento atual
  2. prev:indica que o link é para a parte anterior do documento atual("Preveous" - "anterior")
  3. author:indica que é um link para o site do autor do artigo atual
  4. external:indica que é um link para outro site que não faz parte do site atual
  5. nofollow:indica que é um link para um site não endossado(endossado - que contém a declaração de transferência de posse. No caso não endossado significa que ele não é confiável ou que você não da a repercurssão através do seu site), como um link pago
