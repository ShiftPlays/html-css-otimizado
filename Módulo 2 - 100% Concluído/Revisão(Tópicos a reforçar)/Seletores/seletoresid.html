Aqui vamos praticar no html e nas css os identificadores(id) uma propriedade que nos ajuda a separar cada função com base na organização que estipularmos.
Para ser breve e objetivos com explicações basicamente ao utilizarmos esta propriedade, podemos usar apenas uma vez de acordo com o nome que deixarmos para identificar ela, nesse caso só podemos ter um identificador para cada função que estipularmos.

Prática (id's)

Bom a partir de tudo que eu escrever aqui começando com o que você está lendo agora será uma demonstração de identificadores(id).

Basicamente o que eu fiz aqui foi exatamente isso:

html

Semântica

            
                <p id="demonstração">Bom a partir de tudo que eu escrever aqui começando com o que você está lendo agora será uma demonstração de identificadores(id).</p>

css

Forma

            
                @charset "UTF-8";
                
                p#demonstração {
                    font-family:'Lucida Sans', 'Lucida Sans Regular', 'Lucida Grande', 'Lucida Sans Unicode', Geneva, Verdana, sans-serif
                }

Como a gente já bem sabe, a ordem dessas fontes vão sendo demonstrada de acordo com a capacidade e limitação do navegador. Caso uma fonte não seja identificada pelo navegador a outra será demonstrada e assim sucessivamente até chegar na genérica "sans-serif" caso nenhuma delas sejam identificadas.
