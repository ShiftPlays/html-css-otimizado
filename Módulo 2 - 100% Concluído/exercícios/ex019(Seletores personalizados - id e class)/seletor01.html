# Cerquilha usada para representar a propriedade (id="") do HTML

HTML:

            
                
<h1 id="principal">Seletores Personalizados</h1>
    <h1>Seletores Personalizados 
(Repetido propositalmente apenas para explicarmos a respeito desse tema)</h1>

Em html - "id" (propriedade) = em CSS - "#" (cerquilha)

  • Só pode uma por tag

Em html - "class" (propriedade) = em CSS - "Ponto(.)"
  • Pode ser usado em várias tags

Em outras palavras a propriedade (id="") no html é representada nas CSS como cerquilha(#). Já a propriedade (class="") no html é representada como um ponto(.) nas CSS

Basicamente "id" e "#" - "class" e "ponto (.)" são os responsáveis pelos seletores personalizados e elas podem ser usadas em todas as tags consequentimente refletindo nos seletores.

O (id="") se sobrepõe as configurações de classe (class="") caso queiramos juntar as propriedades (basicamente conseguimos colocar "id" e "class" no mesmo elemento).Também conseguimos colocar mais de uma classe (class="") no mesmo elemento.

Agora qual a diferença de (id="") e (class="")??
A diferença entre as duas são a forma de utilizar, de resto, elas tem basimante a mesma função. A propriedade "id" (id="") conhecida como "identificador" é utilizada apenas uma vez por tag segundo as indicações da W3C (World Wide Web Consortium), já as classes (class="") podem ser usadas mais de uma vez em várias tags de acordo com suas funções para auxiliar melhor na organização do site.
