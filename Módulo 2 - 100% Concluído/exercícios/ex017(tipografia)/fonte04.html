Tipos de alinhamentos

Conceito do seletor "text-align" e suas declarações

"Text-align" em português "alinhamento-de-texto", e seu nome já diz muito sobre a sua função, então sendo bem direto, basicamente são essas as declarações de "text-align".

  • left ("esquerda")
  • right ("direita")
  • center ("centro")
  • justify ("justificar" - ajusta o texto com base no tamanho da aba)

Exemplo:

            
                <style>
                    body {
                        text-align: left;
                    }

                    h1 {
                        text-align: center;
                    }

                    h2 {
                        text-align: right;
                    }

                    p {
                        text-align: justify;
                    }
                </style>

Conceitos do seletor "text-indent" e suas declarações

"Text-indent" em português "recuo-de-texto"(recuo no significado de espaço que fica mais para trás de um alinhamento.), no caso o que chamamos de parágrafo é a função desse seletor, já as declarações desse seletor ajudam no ajuste do espaçamento desse parágrafo.

Exemplo:

            
                <style>
                    p {
                        text-indent: 30px;
                    }
                </style>

Esse é apenas uma declaração para ajusatar o espaçamento para trás do texto(Parágrafo) em pixel(px) 😎👍
